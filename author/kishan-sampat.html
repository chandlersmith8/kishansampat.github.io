Getting your environment ready

Published: Sun 14 June 2020
By Kishan Sampat

In Learning Path.

tags: environment setup

Once you have set up Ubuntu in VirtualBox, the next steps are to install Python, a text editor and any other personal software that you may require for your learning. To begin with, Python and a text editor should suffice.

There are tons of text editors out there, and one could get easily confused with the choices available. I reached out to some of my colleagues for recommendations; Sublime Text & Visual Studio Code seemed to be the general recommendations for beginners. Note that for Windows, you can download the .exe installation files, however, for Ubuntu, you can install them directly via the terminal using sudo commands. You can simply search on Google and you’d be able to find tons of instructions out there on how to do this.

For Python, Ubuntu already comes with a version of Python installed, however, it is best to have the latest one installed. I’d recommend installing it in a Virtual Environment within your working folders. I’ll come to that a bit later, once we’ve started working on some code. Cheers!

Additionally, you also need to install git on your machine. On Ubuntu, you can simply run the following command in your terminal: $ sudo apt install git-all
